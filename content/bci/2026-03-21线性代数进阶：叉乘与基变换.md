## 1. 叉乘（Cross Product）

> [!TIP] 几何意义
>  * **大小**： 平行四边形的 **面积**。
>  * **方向**： 垂直于原平面，遵循 **右手法则** 。`其实我更喜欢右手螺旋法则，左边开始向右边转`

### NumPy 实战
```python
import numpy as np

# 定义 3D向量
v = np.array([1, 0, 0])
w = np.array([0, 1, 0])

# 计算叉乘（np.cross)
cp = np.cross(v, w)
print(f"3D 叉乘结果：{cp}")

# 2D 向量计算（返回面积标量，重点思考叉乘面积和行列式的联系）
v_2d = np.array([2, 0])
w_2d = np.array([0, 3])
area = np.cross(v_2d, w_2d)
print(f"2D 面积:{aera}")
```

## 2. 基变换 (Change of Basis)

> [!QUOTE] 核心哲学 同样的物理位移，在不同的坐标轴下，数字描述不同。 **基变换矩阵 P** 的列，就是对方基向量在“我方”看来坐标。

### 🔄 坐标转换逻辑

1. **对方 -> 我方**： $\vec{v}_{ours} = P \cdot \vec{v}_{theirs}$
    
2. **我方 -> 对方**： $\vec{v}_{theirs} = P^{-1} \cdot \vec{v}_{ours}$

 > [!NOTE] 核心记忆 
 > 矩阵 $P$ 就像是“詹妮弗语言”到“我们语言”的**直接翻译机**。所以 $P$ 直接乘在詹妮弗的向量上。

### 💻 NumPy 实战

Python

```
import numpy as np

# P 矩阵的列是对方的基向量 (詹妮弗的坐标系)
P = np.array([[2, -1], [1, 1]])

# 1. 翻译坐标：将詹妮弗的 [2, 2] 转换成我们的坐标
v_jennifer = np.array([2, 2])
v_ours = P @ v_jennifer
print(f"我们的坐标: {v_ours}")

# 2. 逆翻译：将我们的 [3, 3] 转换成詹妮弗的坐标
v_back = np.linalg.inv(P) @ np.array([3, 3])
print(f"詹妮弗的坐标: {v_back}")
```
