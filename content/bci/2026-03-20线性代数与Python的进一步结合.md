## 1. 矩阵复合 (Matrix Composition)

**直觉**：多个变换叠加。在数学和代码中，执行顺序是**从右往左**。

Python

```
import numpy as np

# A: 旋转矩阵 | B: 剪切矩阵
A = np.array([[0, -1], [1, 0]])
B = np.array([[1, 1], [0, 1]])

# 复合变换：先旋转 A，再剪切 B
# 推荐使用 @ 符号，它比 np.dot() 更直观
C = B @ A 

print("复合后的变换矩阵 C:\n", C)
```

> [!CAUTION] 顺序陷阱 
> $C = B \cdot A$ 意味着对于向量 $\vec{v}$，变换过程是：
 > 1. 先碰到右边的 $A$（执行变换 A）。
 > 2. 结果再碰到左边的 $B$（执行变换 B）。
 > 
 > **在 Obsidian 记成：右边的先动，左边的后动。**


## 2. 行列式与逆矩阵 (Determinant & Inverse)

**直觉**：

- **行列式**：空间面积/体积的缩放比例。
    
- **逆矩阵**：变换的“撤回”操作（Undo）。
    

Python

```
# 计算行列式 (使用 linalg 子模块)
det_C = np.linalg.det(C)

if det_C != 0:
    # 只有行列式不为 0，矩阵才可逆
    C_inv = np.linalg.inv(C)
    print("逆矩阵已生成，可用于撤回变换")
else:
    print("警告：空间已坍缩（降维），不可逆！")
```

## 3. 点积与投影 (Dot Product)

**直觉**：衡量两个向量的“相关性”或“重合度”。

Python

```
v = np.array([3, 4])
u = np.array([1, 0]) # x轴单位向量

# 点积运算
# np.dot 放在根目录下，因为它属于最基础的运算
projection = np.dot(v, u)

print(f"向量 v 在 u 方向上的投影长度: {projection}")
```

##  常用函数速查表

| 功能       | 代码语法                       | 归属模块     | 几何意义     |
| -------- | -------------------------- | -------- | -------- |
| **矩阵乘法** | `A @ B`                    | 根目录      | 变换叠加     |
| **点积**   | `np.dot(u, v)`             | 根目录      | 投影 / 相似度 |
| **行列式**  | `np.linalg.det(A)`         | `linalg` | 空间缩放比例   |
| **逆矩阵**  | `np.linalg.inv(A)`         | `linalg` | 撤回/反转变换  |
| **矩阵秩**  | `np.linalg.matrix_rank(A)` | `linalg` | 有效信息的维度  |

>[!top] 在现代化的表述方法里 dot表点乘，@表矩阵乘法

## BCI 联想

在脑机接口中，我们采集到的 64 通道数据就是一个巨大的矩阵。

1. **点积**用来计算不同脑区信号的相关性。
    
2. **行列式为 0** 通常意味着某些电极短路了，导致数据冗余。
    
3. **逆矩阵**用于从头皮电压推测大脑内部的真实电活动（源定位）。
