## 1.数学素养直觉（3Blue1Brown有感）
> 矩阵不是一堆数字的格子， 而是对空间的“一种操纵方法"。

- **向量 (Vector)**： 空间中的一个箭头，或者一个坐标点。
- **矩阵（Matrix）**：变换空间的规则。
	 - 第一列：代表原始基底 $\vec{i}$  变换后的位置
	 - 第二列：代表原始基地 $\vec{j}$  变换后的位置
	 - 第 n 列：以此列推。
- **线性变化**： 保持网络平行且等距的变换（旋转、拉伸、切变）。

## 2. 代码实现（初步的Numpy实战）
今晚解决了一个`TypeError`，记住了`np.array`定义矩阵需要双层中括号 `[[]]` 。

```python
import numpy as np

#in this step ,I have created a 1-D martrix
v = np.array([1, 2])

#I have create a rotation martrix
rotation = np.array([[0, -1], [1, 0]])

# @ means "turn to" in the video of 3Blue1Brown
result = rotation @ v

print(f"旋转以后的坐标：{result}")
```
